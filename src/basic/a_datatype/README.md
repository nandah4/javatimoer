# Standar Penulisan File Java

- Nama file harus sama dengan nama class public yang ada dalam file.
- Harus menggunakan camel case dalam menulis class ataupun file
- Nama method dan variabel menggunakan camel case (dimulai dengan huruf kecil kemudian besar)
- Nama variable harus menggambarkan isi fungsi variable
- Nama variable konsta (static final) ditulis dengan uppercase dan dipisahkan dengan “ \_ “
